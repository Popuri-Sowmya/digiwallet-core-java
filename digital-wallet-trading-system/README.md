# Concurrent Digital Wallet & Trading System

Core Java backend project implementing a thread-safe wallet and stock trading system using JDBC and PostgreSQL.

## Features

- User management
- Wallet operations (deposit, withdraw, transfer)
- Deadlock-safe money transfers using ReentrantLock
- Stock buy/sell with transactional consistency
- Portfolio tracking
- Stream API analytics
- JUnit tests

## Tech Stack

- Java
- JDBC
- PostgreSQL
- JUnit 5
